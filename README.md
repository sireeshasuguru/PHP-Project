# 📝 PHP Blog App – Task 3 (Advanced Features)

This project is a continuation of a PHP CRUD Blog Application enhanced with advanced features as part of my internship. Built using **PHP**, **MySQL**, **Bootstrap**, and **XAMPP**, it includes login/register authentication, post creation, and now includes **search**, **pagination**, and **UI styling**.

---

## 🚀 Features

✅ **Create, Read, Update, Delete (CRUD)** for blog posts  
🔐 **User Authentication** (Login & Register)  
🔍 **Search Functionality** (by post title/content)  
📄 **Pagination** (5 posts per page)  
🎨 **Bootstrap-enhanced Responsive UI**  
🗃️ Clean folder structure and database separation  

---

## 🛠️ Technologies Used

- **PHP** (Core Logic)
- **MySQL** (Database)
- **Bootstrap 5** (Styling)
- **XAMPP** (Local Server Environment)
- **HTML/CSS** (Frontend)
